# Student Information System

## Description
This is a simple Java console application that collects student information and displays a summary, including a performance evaluation based on the student's CGPA.

## Features
- Enter student name
- Enter student age
- Enter student department
- Enter student CGPA
- Validates age (16–100)
- Validates CGPA (0.0–4.0)
- Prevents empty names
- Displays a performance evaluation

## Technologies Used
- Java
- IntelliJ IDEA
- Git
- GitHub

## How to Run
1. Open the project in IntelliJ IDEA.
2. Run the `StudentInformationSystem.java` file.
3. Enter the requested information.
4. View the student summary.
